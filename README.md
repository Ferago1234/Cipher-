# ZIPHER v3 - Strategic Intelligence System

> Numerology. Astrology. Chinese Zodiac. Mindfulness. AI-powered affirmations.

## What's New in v3
- Mobile-responsive layout with hamburger navigation
- Dark / Light mode toggle
- Mindfulness tab: 6 evidence-based practices (MBSR, CBT, ACT, TM, Vipassana, Somatic)
- 30-Day AI Affirmation Generator (Gratitude, Empathy, Love, Forgiveness) personalised to your numerology/astrology
- Terms of Service agreement screen (required before app access)
- Footer with disclaimer and legal notices

## Quick Start
```bash
npm install
npm run dev
```

## Deploy to Vercel
Push to GitHub. Connect to Vercel. Framework: Vite. Output: dist.
